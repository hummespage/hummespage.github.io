# Hummes Portfolio (GitHub Pages)

Este pacote já vem **pronto** (HTML/CSS/JS) no estilo da tua marca: off-white + pastel + cards.

## Como subir no GitHub Pages (repositorio `SEUUSUARIO.github.io`)
1. Apague tudo do repositório
2. Envie os arquivos deste ZIP para a raiz do repo
3. Settings → Pages → Deploy from a branch → `main` → `/(root)`
4. Pronto: https://SEUUSUARIO.github.io

## Onde trocar infos (se quiser)
- `index.html` (texto/links)
- `script.js` (listas de skills/projetos/timeline)
- `styles.css` (visual)

Obs: o site é estático, então o formulário só copia mensagem (não envia).
